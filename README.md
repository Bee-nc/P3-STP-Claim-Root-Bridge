# P3-STP-Claim-Root-Bridge
Repositorio individual para la práctica P3: ataque STP Claim Root Bridge en la asignatura Seguridad de Redes.   El ataque envía BPDUs falsos para reclamar ser el root bridge en la red, forzando al switch a cambiar su topología y permitiendo al atacante manipular el flujo de tráfico.
